# 00016 — The Static

**Status:** Built
**Last updated:** 2026-05-23

## What it is

A television showing static. Nothing else. No interaction. No explanation.

## How it works

Static noise rendered client-side — no WebSocket, no server calls. The visual is the entire room. SAP institutional aesthetic outside the TV; the static itself is the only content.

None.
