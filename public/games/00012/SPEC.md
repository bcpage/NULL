# 00012 — The Yellow Door

**Status:** Built
**Last updated:** 2026-05-23

## What it is

A yellow door. It does not open.

## How it works

Static room. No server calls, no WebSocket, no interaction. The door is rendered in CSS or canvas. There is nothing to do here. SAP institutional aesthetic.

## Data files

None.

## Navigation

Matrix: down → 00011 (Number Dispenser), right → 00013 (Cowsay). Both bidirectional.

## Locked content

None. The door itself is the locked content.

## Notes for future development

The door not opening is the entire point. Any mechanic that opens it would undermine the room. If a secret unlock condition is ever added, it should feel impossible to find.
