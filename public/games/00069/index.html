CATEGORY C — DOCUMENT OF A PREVIOUS VERSION / will not be updated
This room documents an earlier version of the NULL facility. The information below was accurate at the time of writing. It has not been updated and will not be updated. Some features described here no longer exist. Some rooms described here were never built. The discrepancies are intentional.
Platform overview — as of initial deployment

NULL is a 14-room web facility running on a shared hosting plan. Players enter through the welcome screen and navigate using left and right arrow buttons. There is no matrix. Navigation is linear.

The facility tracks visitors by IP address. There is no cookie system. Visitors cannot be distinguished across sessions.

Navigation system — original

Navigation was sequential: ◀ and ▶ buttons moved between rooms in order. There was no matrix. There were no directional connections. You could not reach a room by going "up." The concept of "up" did not exist in this facility.

The matrix navigation system was introduced in a later version. The rooms that existed before the matrix were retrofitted into it. Not all of them translated cleanly.
