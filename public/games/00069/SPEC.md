# [00069] — Useless Information C (Fossil Room)

**Status:** Built
**Last updated:** 2026-05-23

## What it is
Documents an earlier version of the NULL facility as if the observer is reading
a historical record. The original platform had 14 rooms, sequential navigation
(no matrix), and IP-based tracking (no cookies). Several original rooms no longer
exist. Several planned features were never built. The room acknowledges its own
obsolescence and states it will not be updated.

## How it works
Static HTML. Strikethrough CSS on removed features. No interaction. No server calls.
The content is internally consistent with the platform's real history (approximately).

## Notes for future development
Do not update the content to reflect current platform state — that defeats the purpose.
If the platform changes significantly, a new Fossil Room documenting that version
is appropriate. This one documents the version that existed before the matrix.
