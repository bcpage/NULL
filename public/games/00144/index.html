The Paradox (Plutarch, ~75 AD):

The Athenians preserved the ship in which Theseus returned from Crete, replacing old planks with new ones as they decayed. Philosophers debated whether the repaired ship was the same object as the original — which is now called the Ship of Theseus paradox. Thomas Hobbes added a second complication: if the old planks were collected and reassembled into a second ship, which would be the original?

What it reveals:

The paradox exposes that "sameness" is not a property of objects — it is a judgment made by observers. The ship does not know whether it is the same ship. The question is about language and identity, not about wood.
